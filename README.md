# Creative Developer Portfolio

A premium, responsive personal portfolio website built with HTML, CSS, and plain JavaScript. It features a modern dark aesthetic, interactive elements, and a focus on user experience.

## Features

- **Modern Design**: Gradient text, glassmorphism effects, and a custom cursor.
- **Theme System**: Toggles between Dark (default) and Light modes, persisting user preference.
- **Interactive Elements**:
  - Typing text effect in the Hero section.
  - Smooth scrolling navigation.
  - Project cards with hover animations.
- **Contact Form**: Interactive form UI with success feedback state.
- **Responsive**: Fully responsive layout that adapts to mobile, tablet, and desktop screens.

## Project Structure

- `index.html`: The main HTML structure.
- `style.css`: All styles, including CSS variables for theming and animations.
- `script.js`: Logic for the theme toggle, typing effect, mobile menu, and scroll observer.

## Customization

1.  **Update Content**: Edit the text in `index.html` to match your personal details.
2.  **Images**: Replace the CSS gradient placeholders in `.project-image` with actual `<img>` tags or background images of your work.
3.  **Links**: Update the `href` attributes in the social links and project cards.

## Usage

Simply open `index.html` in any modern web browser. No build step or server is required.
